# Design Asset Structure

Put design assets under `assets/design/pages/` grouped by page.

## Pages

- `onboarding`
- `status`
- `world`
- `log`
- `log_detail`

## Suggested folders per page

- `screens/`: full-page PNG exports from Figma
- `icons/`: back, menu, heart, nav, arrows, etc.
- `characters/` or `avatars/`: people, penguin, profile art
- `scenes/`: room and environment illustrations
- `gifts/`: gift and item art
- `components/`: buttons, cards, sliders, borders, textures
- `spec/`: size notes, color values, spacing screenshots, font notes

## Example

`assets/design/pages/onboarding/screens/init.png`

`assets/design/pages/world/scenes/bedroom_a.png`

`assets/design/pages/log_detail/spec/colors.txt`
