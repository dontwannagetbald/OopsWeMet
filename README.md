「Oops! We met」是一种面向日常公共空间的低压力社交媒介。用户将自己的性格、兴趣与社交偏好映射为一个随身智能体,
并将其嵌入便携式硬件中。当两位用户在空间中短暂接近时,彼此的智能体会先行感知对方、交换信息,并在后台进行轻量化的
社交互动。人与人无需立即开口,社交关系在不经意间悄然萌芽,呈现出偶然、隐秘而低干扰的交流体验。
